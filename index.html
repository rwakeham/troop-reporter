Where to get the files

  1. Rank Requirements Status — In TroopWebHost, Menu → Advancement → Maintain Advancement → Export Rank Requirements to Excel.
  2. Merit Badges — In TroopWebHost, Menu → Advancement → Maintain Advancement → Merit Badges, then export all results.

Both files cover the full troop. CSV and XLSX are both supported. You can drop one or both files; the dashboard uses whatever it has.
